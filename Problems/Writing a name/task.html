Writing a name Open on JetBrains Academy

You are given a list containing letters of a name. Print them out as a string with these chars separated by hyphens and with an exclamation point in the end. Do so using arguments of the print() function!

So, if the letters in the list name are ['K', 'A', 'T', 'E'], your program should print the string 'K-A-T-E!'
